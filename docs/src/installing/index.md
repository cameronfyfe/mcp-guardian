# Installing

MCP Guardian can be installed directly from [Github releases](https://github.com/eqtylab/mcp-guardian/releases). Go to the section for your operating system for detailed instructions.

* [Linux](./linux.md)
* [MacOS](./macos.md)
* [Windows](./windows.md)
